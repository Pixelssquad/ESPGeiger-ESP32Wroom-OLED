# ESPGeiger

ESPGeiger is a project for collecting, calculating and reporting from Geiger counters with a pulse output signal. The firmware is written to be installed onto the common esp8266 and esp32 boards.

## Outputs
- MQTT
- ThingSpeak
- Radmon.org
- gmcmap.com

Planned:
- ESPGeiger API
- Async Webserver
- Display support
